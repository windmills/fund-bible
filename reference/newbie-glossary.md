# 新手术语百科

用户是投资小白。每个专业术语首次出现时必须用大白话解释。本文档是完整术语库，按类别整理。

---

## 一、技术指标类

### 均线/MA（Moving Average）

**大白话**：过去 N 天基金净值的平均值连成的线。

**比喻**：就像人的"平均体重"——MA5 是最近 5 天的平均净值（一周），MA20 是最近 20 天的（一个月），MA60 是最近 60 天的（一个季度）。

**怎么看**：
- 净值在均线之上 = 趋势向上（好）
- 净值跌破均线 = 趋势可能变坏
- 多条均线从上到下排成 MA5>MA10>MA20>MA60 = "多头排列"（最强买入信号）

### 金叉/死叉

**大白话**：两条均线的交叉点。

**比喻**：金叉 = 短期均线从下往上穿过长期均线 = "加速信号"（好）。死叉 = 短期从上往下穿 = "减速信号"（不好）。

### 多头排列/空头排列

**大白话**：所有均线按短期在上、长期在下的顺序排列。

**比喻**：
- 多头排列 = MA5>MA10>MA20>MA60，全部向上 = 所有人都在赚钱 = 满仓持股
- 空头排列 = MA5<MA10<MA20<MA60，全部向下 = 所有人都在亏钱 = 空仓观望

### MACD（指数平滑异同移动平均线）

**大白话**：判断趋势强弱的指标。

**比喻**：MACD 是价格的"加速度"——净值还在涨但 MACD 红柱缩短 = 虽然还在涨但速度在减慢（该警惕了）。

**组成**：
- DIF（快线）：短期趋势
- DEA（慢线）：长期趋势
- MACD 柱：DIF 和 DEA 的差值

**怎么看**：
- 红柱（柱>0）= 多头力量强 = 还在加速上涨
- 绿柱（柱<0）= 空头力量强 = 还在加速下跌
- 红柱变短 = 多头在减弱 = 准备减仓
- 绿柱变短 = 空头在减弱 = 准备反弹

### 顶背离/底背离（MACD）

**大白话**：价格和指标"唱反调"。

**比喻**：
- 顶背离 = 净值创新高，但 MACD 没创新高 = "表面很强，实际虚了"= 该卖了
- 底背离 = 净值创新低，但 MACD 没创新低 = "表面很弱，实际在蓄力"= 该关注了

**这是 MACD 最重要的信号，准确率很高。**

### KDJ（随机指标）

**大白话**：判断超买超卖的短线指标。

**比喻**：KDJ 就像弹簧——压得太紧（K<20，超卖）会弹回来，拉得太长（K>80，超买）会缩回去。

**组成**：K 线、D 线、J 线（J 是 K 和 D 的加速版）

**怎么看**：
- K<20 = 超卖（跌太多了，可能反弹）→ 关注买入
- K>80 = 超买（涨太多了，可能回调）→ 警惕卖出
- K 上穿 D（金叉）且 K<20 = 低位金叉 = 买入信号

**注意**：KDJ 在单边趋势中会"钝化"（长时间停在超买/超卖区），此时 KDJ 失效。

### RSI（相对强弱指标）

**大白话**：衡量多空力量强弱的温度计。

**比喻**：RSI 就像情绪温度计——>70=太热了（超买），<30=太冷了（超卖）。

**怎么看**：
- RSI > 70 = 超买（可能要回调）
- RSI < 30 = 超卖（可能要反弹）
- RSI 顶背离（净值新高但 RSI 没新高）= 上涨乏力
- RSI 底背离（净值新低但 RSI 没新低）= 蓄力反弹

### BOLL（布林带）

**大白话**：给净值画一个"走廊"，上轨是天花板，下轨是地板。

**比喻**：就像乒乓球在房间里弹——碰到天花板会弹下来，碰到地板会弹上去。中轨是房间的中央。

**基金专用**：基金波动小，用 **1.5 倍标准差**（股票用 2 倍，太宽松了）。

**组成**：
- 上轨（UP）= 天花板 = 中轨 + 1.5 × 标准差
- 中轨（MB）= 地面 = 20 日均线
- 下轨（DN）= 地板 = 中轨 - 1.5 × 标准差

**怎么看**：
- 净值碰到上轨 = 可能太高了，警惕回调
- 净值碰到下轨 = 可能太低了，关注反弹
- 带宽收口（走廊变窄）= 要变盘了（选择方向）
- 收口后向上开口 = 方向选了向上 → 跟进
- 收口后向下开口 = 方向选了向下 → 离场

### WR（威廉指标 %R）

**大白话**：当前净值在近期波动范围中的位置。

**比喻**：就像电梯——WR 高=在一楼（超卖），WR 低=在顶楼（超买）。

**注意方向和 KDJ/RSI 相反**：WR > 80 = 超卖，WR < 20 = 超买。

### CCI（顺势指标）

**大白话**：净值偏离平均值的程度。

**怎么看**：
- CCI > +100 = 超买区
- CCI < -100 = 超卖区
- CCI 上穿 -100 = 超卖回升 = 买入信号
- CCI 下穿 +100 = 超买回落 = 卖出信号

### BIAS（乖离率）

**大白话**：净值离均线有多远。

**比喻**：就像橡皮筋——拉得太长（乖离率大）会弹回来。

**怎么看**：
- BIAS > 5%（6日）= 短期涨太多 = 警惕回调
- BIAS < -5%（6日）= 短期跌太多 = 关注反弹

### DMI（趋向指标）

**大白话**：判断趋势方向和强度的指标。

**组成**：
- +DI = 多头力量
- -DI = 空头力量
- ADX = 趋势强度（不分方向）

**怎么看**：
- +DI > -DI = 多头占优
- ADX > 25 = 趋势明显（值得跟随）
- ADX < 20 = 没有趋势（震荡市，别追涨杀跌）

### SAR（抛物线转向）

**大白话**：自动止损/止盈的指标。

**比喻**：SAR 就像安全网——在净值下方时（红色）保护你持有，翻到上方时（绿色）提醒你离场。

### OBV（能量潮）

**大白话**：累积净值的涨跌方向，反映资金流向。

**怎么看**（仅 ETF 适用）：
- OBV 上升 + 净值上升 = 量价配合，健康
- OBV 下降 + 净值上升 = 量价背离，虚涨

## 二、专业指标类

### Alpha（阿尔法）

**大白话**：基金经理的"真本事"——跑赢市场的那部分收益。

**比喻**：全班平均考 80 分，你考了 85 分，多出的 5 分就是你的 Alpha。如果大盘涨了 10%，你的基金也涨了 10%，这不叫本事（只是跟着大盘），多出来的部分才是 Alpha。

**怎么看**：
- Alpha > 0 = 基金经理有选股能力 = 好
- Alpha < 0 = 跑不赢大盘 = 不如直接买指数基金

### Beta（贝塔）

**大白话**：基金跟大盘的"跟风程度"。

**比喻**：Beta 就像两个人坐过山车——Beta=1.2 的人晃得更厉害（大盘涨 1%，他涨 1.2%），Beta=0.8 的人晃得更轻。

**怎么看**：
- Beta > 1 = 高弹性（涨跌比大盘猛）= 牛市好，熊市惨
- Beta < 1 = 低弹性（涨跌比大盘稳）= 熊市避风港
- Beta = 1 = 跟大盘同步

### 夏普比率（Sharpe Ratio）

**大白话**：每冒一分险，换来几分赚头。

**比喻**：就像打工的"性价比"——有人 996 赚 1 万，有人朝九晚五也赚 1 万，后者的"夏普比率"更高。

**公式**：夏普 = (年化收益 - 无风险利率) / 年化波动率

**怎么看**：
- > 1.5 = 优秀（冒 1 份险赚 1.5 份钱）
- 1.0-1.5 = 良好
- < 0 = 冒了险还亏钱 = 不值得

### 索提诺比率（Sortino Ratio）

**大白话**：夏普比率的改良版——只把"坏波动"（下跌）当风险，上涨不算风险。

**比喻**：夏普比率把涨和跌的波动都算风险（不合理），索提诺只算跌的波动（更合理）。

**怎么看**：通常比夏普比率高，> 2 算优秀。

### 卡玛比率（Calmar Ratio）

**大白话**：年化收益 / 最大回撤。衡量"用最小回撤换最大收益"。

**比喻**：如果你赚了 30% 但中间最多亏过 20%，卡玛 = 30/20 = 1.5。赚得多且回撤小 = 卡玛高 = 好。

### 信息比率（Information Ratio）

**大白话**：主动管理能力的"照妖镜"——偏离基准去押注，值不值得？

**比喻**：基金经理主动偏离指数去选股，如果每次偏离都赚了钱（信息比率高），说明是真有水平；如果偏离了还亏钱（信息比率低），说明是瞎折腾。

**怎么看**：> 0.5 算不错，> 1 算优秀。

### 跟踪误差（Tracking Error）

**大白话**：指数基金净值和基准指数的偏差。

**比喻**：就像复印——好的指数基金是"高清复印"（跟踪误差小），差的是"模糊复印"（跟踪误差大）。

**怎么看**：< 2% = 优秀（仅限指数基金）

### 最大回撤（Maximum Drawdown）

**大白话**：最惨的时候会亏多少。

**比喻**：净值从 10 块跌到 6 块，最大回撤就是 40%。就像问你：如果买在最高点、卖在最低点，最多亏多少？

**怎么看**：
- < 10% = 低风险
- 10-20% = 中等风险
- 20-30% = 高风险
- > 30% = 极端风险

### 年化波动率

**大白话**：净值的震荡幅度。

**比喻**：波动率高 = 像坐过山车（大起大落），波动率低 = 像坐高铁（平稳）。

### 换手率

**大白话**：基金经理每年把手里股票换掉多少。

**比喻**：换手率 300% = 一年内手里的股票全换了 3 遍 = 频繁交易型。换手率 100% = 一年换一遍 = 正常。换手率 < 100% = 换得还没一年一遍 = 长期持有型。

**怎么看**：
- > 500% = 频繁交易，风格激进，警惕
- 100-300% = 正常水平
- < 100% = 长期持有，风格稳健

### 回撤修复时间

**大白话**：从谷底涨回峰值需要多久。

**比喻**：净值从 10 块跌到 7 块，花了 6 个月才涨回 10 块——回撤修复时间就是 6 个月。

## 三、基本面类

### 市盈率（PE）

**大白话**：你花多少钱买 1 块钱的利润。

**比喻**：PE = 20 意味着你花 20 块买了一个每年赚 1 块钱的公司。PE 越低越便宜（但要看行业）。

**怎么看**：
- PE < 15 = 低估值（便宜）
- PE 15-40 = 合理估值
- PE > 80 = 高估值（贵，有泡沫风险）

### 市净率（PB）

**大白话**：你花多少钱买 1 块钱的净资产。

**比喻**：PB = 2 意味着你花 2 块钱买了账面价值 1 块钱的资产。

### 持仓集中度

**大白话**：前十大重仓股占基金资产的比例。

**比喻**：集中度 60% = 60% 的鸡蛋放在 10 个篮子里，剩下 40% 放在无数个小篮子里。

### 机构持有比例

**大白话**：专业机构（银行/保险/养老金）持有这只基金的比例。

**比喻**：机构持有比例高 = 专业机构也看好这只基金 = "聪明钱"的认可。但太高（>80%）也不好，机构一跑净值就崩。

### 风格漂移

**大白话**：基金经理突然改变投资风格。

**比喻**：本来是"价值投资者"（买低估值股票），突然变成"追热点"（买热门概念股）= 风格漂移 = 策略不稳定。

### 规模魔咒

**大白话**：基金规模暴增后业绩通常下滑。

**为什么**：钱太多了，好股票数量有限，基金经理被迫买次优选择。

## 四、宏观类

### 美联储（Federal Reserve / Fed）

**大白话**：美国的"央行"，决定美元利率。

**比喻**：美联储就像水龙头——加息=关水（钱变少，市场跌），降息=开水（钱变多，市场涨）。

### FOMC（联邦公开市场委员会）

**大白话**：美联储的决策机构，每年开 8 次会决定利率。

### 加息/降息

**大白话**：
- 加息 = 提高利率 = 借钱更贵 = 企业成本增加 = 股市通常跌
- 降息 = 降低利率 = 借钱更便宜 = 企业成本降低 = 股市通常涨

### 美元指数

**大白话**：美元相对于其他货币的强弱。

**比喻**：美元指数上升 = 美元变"值钱"了 = 人民币相对贬值了 = QDII 基金汇率赚钱。

### 汇率损益

**大白话**：因为汇率变化导致的额外赚/亏。

**举例**：你买了纳斯达克 QDII，纳斯达克涨了 10%。但同期美元贬值了 3%（人民币升值），你的实际收益 ≈ 10% - 3% = 7%。汇率"吃掉"了 3%。

### QDII 额度

**大白话**：国家给基金公司的"海外投资配额"。

**比喻**：就像出国旅游的外汇额度——每人每年 5 万美元。QDII 额度用完了，基金就会"限购"（暂停申购）。

## 五、交易机制类

### T+1 / T+2

**大白话**：交易确认的时间。

**比喻**：T+1 就像网购——今天下单（T 日），明天发货（确认份额），后天到货（看到盈亏）。

**具体规则**：
- 场外基金：T+1 确认（今天 15:00 前买入，明天确认份额）
- QDII 基金：T+2 确认（比国内基金慢一天）
- 赎回到账：货币基金 T+0/T+1，股票型 T+3-T+7

### 申购费/赎回费

**大白话**：
- 申购费 = 买入时交的"入场费"（通常 0.15%-1.5%，很多平台打 1 折）
- 赎回费 = 卖出时交的"离场费"（持有越久越便宜，<7 天有 1.5% 惩罚性费率）

### 管理费/托管费

**大白话**：
- 管理费 = 每年给基金公司的"管理工资"（股票型通常 1.5%/年）
- 托管费 = 每年给托管银行的"保管费"（通常 0.25%/年）

**这些费用每天自动从净值里扣除，你感觉不到，但长期累积很可观。**

### A 类 / C 类份额

**大白话**：同一只基金的两种"收费模式"。

**比喻**：
- A 类 = "年卡"：进门交一笔申购费（0.15%），之后每年只交管理费。**适合长期持有（>1年）**
- C 类 = "月卡"：进门免费（无申购费），但每年多交一笔销售服务费（0.4-0.8%）。**适合短期持有（<1年）**

**怎么选**：
- 打算持有 > 1 年 → 选 A 类
- 打算持有 < 1 年 → 选 C 类
- 不确定 → 选 A 类（长期更划算）

### 销售服务费

**大白话**：C 类份额每年额外收取的费用，替代申购费。

### 惩罚性赎回费

**大白话**：持有 < 7 天就卖出，收 1.5% 的惩罚性费用。

**为什么**：防止短期投机冲击基金运作。**买基金不要 7 天内卖出！**

### LOF（上市型开放式基金）

**大白话**：既能在场外（银行/平台）申赎，又能在场内（股票账户）买卖的基金。

### ETF 联接基金

**大白话**：场外版的 ETF——你不炒股没有股票账户，但想买 ETF，就买 ETF 联接基金。

### 溢价率/折价率

**大白话**：场内交易价格和实际净值的偏差。

**比喻**：
- 溢价 = 交易价格 > 净值 = 你花更多钱买了实际不值那么多的东西（亏了）
- 折价 = 交易价格 < 净值 = 你花更少钱买了实际值更多的东西（赚了）

## 六、资金面类（主力动向）

### 主力资金

**大白话**：大机构（基金/保险/社保/QFII）的资金，它们钱多，买卖能影响价格。

**比喻**：主力就像鲸鱼，散户就像小鱼。鲸鱼往哪游，海水就往哪涌。

### 超大单/大单/中单/小单

**大白话**：按每笔成交金额分类。

| 分类 | 金额范围 | 谁 |
|------|----------|-----|
| 超大单 | > 100万 | 大机构/主力 |
| 大单 | 20-100万 | 中型机构 |
| 中单 | 4-20万 | 大户 |
| 小单 | < 4万 | 散户（你和我） |

### 主力吸筹

**大白话**：主力悄悄买入，尽量不引起注意。

**比喻**：就像大鲸鱼悄悄吞虾米——一口一口慢慢吃，不让虾米群惊慌逃散。

**怎么识别**：主力净流入 > 0 且散户净流出（散户在卖，主力在接）。

### 主力派发

**大白话**：主力悄悄卖出，把筹码甩给散户。

**比喻**：就像超市打折清仓——表面繁荣（净值在涨），实际是老板在清库存跑路。

**怎么识别**：主力净流出 且散户净流入（散户在追涨接盘，主力在套现）。

### 主力拆单

**大白话**：主力把一笔大买单拆成几百笔小单，伪装成散户在买。

**为什么**：如果直接挂大单买入，会推高价格（买不到便宜价）。拆成小单可以慢慢吸筹不引人注意。

**怎么识别**：大量金额相同或相近的中小单密集出现（DG-1 信号）。

### 对倒

**大白话**：主力自己买自己卖，制造成交量假象。

**比喻**：就像一个人左手卖右手买——交易量看起来很大，实际没有新资金进出。

**目的**：吸引散户注意（"哇，成交这么活跃，肯定有利好"），引诱散户跟风买入。

**怎么识别**：买卖双向同时出现大量中小单（DG-2 信号）。

### 虚假托单/压单

**大白话**：
- 虚假托单 = 挂一个大买单在下面（但不成交），制造"有人撑着"的假象 → 诱多
- 虚假压单 = 挂一个大卖单在上面（但不成交），制造"涨不动"的假象 → 诱空

**比喻**：就像 scarecrow（稻草人）——看着吓人，实际是假的。主力挂上去就是吓你买卖，等他目的达到就撤单。

### 尾盘拉升/砸盘

**大白话**：
- 尾盘拉升 = 收盘前突然大单买入，把价格拉高 → "非奸即盗"（制造好看的收盘价，明天好出货）
- 尾盘砸盘 = 收盘前突然大单卖出，把价格砸低 → 主力急了，不顾成本出逃
