# 宏观面分析指南

本文档详细说明宏观面分析的方法论和数据来源。宏观面分析是基金分析的第五层，权重 10%。

---

## 一、美联储利率政策

### 为什么重要

美联储（美国联邦储备委员会）的利率决策影响全球资本市场：
- **加息**→ 美元升值，全球资金回流美国，新兴市场承压
- **降息**→ 美元贬值，全球流动性增加，风险资产上涨

对基金的影响：
- QDII 基金：直接影响（美元汇率 + 美股/美债价格）
- A 股基金：间接影响（全球风险偏好传导）
- 债券基金：强相关（利率与债券价格反向）

### 数据获取

```
web search "美联储最新利率决议" 或 "Federal Reserve interest rate decision"
```

### 判断当前周期

| 状态 | 特征 | 对基金影响 |
|------|------|-----------|
| 加息周期 | 连续多次加息 | 美元升值→QDII汇率收益；美债价格跌→美债QDII减分 |
| 降息周期 | 连续多次降息 | 美元贬值→QDII汇率损失；美债价格涨→美债QDII加分 |
| 维持不变 | 暂停加息/降息 | 政策真空期，影响中性 |
| 鹰派（偏紧） | 官员表态偏加息 | 提前反应，市场承压 |
| 鸰派（偏松） | 官员表态偏降息 | 提前反应，市场上涨 |

### 超预期事件

- **超预期加息**（市场没预料到）→ 市场恐慌，短期利空 → 减仓避险
- **超预期降息**（市场没预料到）→ 市场狂欢，短期利好 → 加仓

### 对不同基金的影响矩阵

| 美联储状态 | 美股QDII | 美债QDII | A股基金 | 黄金基金 |
|-----------|----------|----------|---------|---------|
| 加息 | 汇率+，估值-（偏多） | 汇率+，价格-（偏空） | 间接-（偏空） | 偏空（美元涨黄金跌） |
| 降息 | 汇率-，估值+（偏多） | 汇率-，价格+（偏多） | 间接+（偏多） | 偏多（美元跌黄金涨） |

## 二、美元指数

### 为什么重要

美元指数衡量美元对一篮子货币的汇率。对 QDII 基金的实际收益有直接影响：

**汇率损益可达 5-10%！** 即使海外资产本身没涨没跌，汇率变化也会造成盈亏。

**公式**：QDII实际收益 = 资产涨跌 + 汇率变化

- 美元升值（指数上升）→ 人民币贬值 → QDII 汇率收益（同样多的美元换成更多人民币）
- 美元贬值（指数下降）→ 人民币升值 → QDII 汇率损失

### 数据获取

```
https://quote.eastmoney.com/gb/ZSDINIW.html
```

### 关键阈值

| 美元指数 | 状态 | 对QDII影响 |
|----------|------|-----------|
| > 110 | 极端高位 | 可能见顶，警惕汇率反转 |
| 100-110 | 强势区 | QDII 汇率收益，加分 |
| 90-100 | 正常区 | 汇率影响中性 |
| < 90 | 极端低位 | 可能见底，关注汇率反弹 |

### 汇率损益计算示例

```
买入纳指QDII时：1美元 = 7.2人民币
当前汇率：      1美元 = 7.0人民币（美元贬值）

纳斯达克指数涨幅：+10%
汇率变化：7.0/7.2 - 1 = -2.8%

实际收益 ≈ 10% - 2.8% = 7.2%（汇率吃掉了2.8%收益）
```

### 分析方法

对 QDII 基金，需要对比基金净值走势和对应海外指数走势：
- 基金涨/指数涨 → 正常
- 基金不涨/指数大涨 → 跟踪偏差，关注
- 基金跌/指数涨 → 异常，需排查原因

## 四、A 股大盘环境

### 数据获取

```
上证指数K线API:
https://push2his.eastmoney.com/api/qt/stock/kline/get?secid=1.000001&fields1=f1,f2,f3,f4,f5,f6&fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61&klt=101&fqt=1&end=20500101&lmt=60
```

### 大盘状态判定

| 大盘状态 | 判定条件 | 对A 股基金操作 |
|----------|----------|---------------|
| 牛市（多头） | 上证 MA5>MA10>MA20>MA60 | 正常操作，积极做多 |
| 震荡 | 均线交叉混合，无明确方向 | 精选个基，轻仓 |
| 弱势 | 上证跌破 MA20 | 减半仓，谨慎 |
| 熊市（空头） | 上证跌破 MA60 | 空仓观望，禁止开新仓 |
| 暴跌 | 单日跌幅 > 3% | 系统性风险，所有 A 股基金减仓 |

### 大盘过滤规则

**大盘不好，个基难搞。系统性风险下，所有技术形态失效。**

1. 大盘多头 → 正常操作
2. 大盘横盘 → 轻仓精选
3. 大盘跌破 MA20 → 减半仓
4. 大盘跌破 MA60 → **空仓，禁止开新仓**

## 五、地缘政治与新闻面

### 数据获取

```
web search "{相关主题} 最新消息" 或 "{行业} 政策 最新"
```

### 影响等级

| 事件类型 | 影响范围 | 持续时间 | 操作 |
|----------|----------|----------|------|
| 中美贸易摩擦 | 科技/QDII 基金 | 中长期 | 减仓相关 |
| 行业重大利好 | 对应行业基金 | 中短期 | 加仓相关 |
| 行业重大利空（监管/制裁） | 对应行业基金 | 中长期 | 减仓相关 |
| 突发黑天鹅（疫情/战争） | 全局 | 短期恐慌 | 降低整体仓位 |
| 汇率政策变动 | QDII | 中期 | 重新评估汇率影响 |

### 新闻解读原则

1. **区分短期情绪 vs 长期趋势**：单次利空不改变长期趋势
2. **看政策方向**：政策连续性比单次事件更重要
3. **关注预期差**：市场已预期的利空影响小，超预期的利空影响大
4. **不追涨杀跌**：新闻驱动的行情往往短期剧烈但不可持续
