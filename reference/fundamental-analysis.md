# 基本面分析指南（晨星五支柱 + 持仓分析）

本文档详细说明基金基本面分析的 methodology。基于晨星（Morningstar）五星评级体系。

---

## 晨星五支柱概述

Morningstar 通过五个维度评估基金，这五个支柱被公认为最权威的基金评估框架：

| 支柱 | 英文 | 核心问题 | 权重 |
|------|------|----------|------|
| People | 基金经理 | 谁在管这只基金？能力强不强？ | 高 |
| Parent | 基金公司 | 公司整体实力如何？ | 中 |
| Process | 投资策略 | 策略清晰吗？可重复吗？ | 高 |
| Performance | 业绩 | 历史业绩好不奶？（低权重） | 低 |
| Price | 费率 | 收费贵不贵？ | 中高 |

**核心理念：过去业绩不代表未来。People + Process + Price 比历史业绩更重要。**

## 一、People 基金经理评估

### 数据来源

```
https://fundf10.eastmoney.com/jjjl_{code}.html
```

### 评估维度

#### 1. 任职期限

| 任职时间 | 评价 | 信号 |
|----------|------|------|
| > 5 年 | 经验丰富，经过完整牛熊周期 | 正面 |
| 3-5 年 | 经验较好 | 正面 |
| 1-3 年 | 任职较短，业绩参考有限 | 中性 |
| < 1 年 | 新手风险 | 谨慎 |

#### 2. 历史业绩

查看经理管理的所有基金的历史业绩排名：

- 管理的基金同类排名前 25% → 优秀
- 管理的基金同类排名 25-50% → 中上
- 管理的基金同类排名后 50% → 较差

#### 3. 管理规模

- 管理规模突然暴增（>200%）→ "规模魔咒"，业绩可能下滑
- 同时管理 > 10 只基金 → 精力分散
- 管理规模适中（10-200 亿）→ 最佳

#### 4. 变更频率

- 近 2 年更换 >= 2 次 → 策略不稳定
- 基金经理突然更换 → 必须重新评估（铁律 3）

#### 5. 投资风格

从经理管理的其他基金和季度持仓变化判断：
- 价值型/成长型/均衡型
- 长期持有型/频繁交易型
- 风格是否稳定（是否漂移）

---

## 二、Parent 基金公司评估

### 评估维度

#### 1. 公司整体实力

- 管理规模行业排名前 10 → 大公司，资源多，稳定性加分
- 旗下同类基金整体排名前 20% → 投研团队强
- 有知名明星基金经理 → 加分

#### 2. 治理合规

- 近 1 年有重大违规/处罚 → 治理风险，建议避开
- 公司高管频繁变动 → 不稳定

#### 3. 产品线

- 产品线丰富 → 综合实力强
- 在特定领域有优势（如某公司的指数基金特别强）→ 领域加分

## 三、Process 持仓分析

### 数据来源

```
https://fundf10.eastmoney.com/ccmx_{code}.html   # 持仓明细
https://fundf10.eastmoney.com/hyzl_{code}.html   # 行业配置
https://fundf10.eastmoney.com/zcpz_{code}.html   # 资产配置
```

### 评估维度

#### 1. 持仓集中度

| 前十大持仓集中度 | 评价 | 含义 |
|------------------|------|------|
| > 60% | 高集中度 | 高风险高收益，波动大 |
| 30-60% | 中等 | 正常 |
| < 30% | 分散 | 风险较低，适合核心配置 |

**比喻：集中度就像鸡蛋放几个篮子——60%集中度相当于6成鸡蛋在一个篮子里。**

#### 2. 行业分布

- 重仓行业 > 5 个 → 分散配置，风险分散
- 重仓集中在 1-2 个行业 → 行业押注，行业景气时大赚，衰退时大亏

**重要：如果用户持有多只基金都是同一行业（如用户当前持有2只CPO基金），需要提醒集中度风险。**

#### 3. 重仓股估值

| 重仓股平均 PE | 估值水平 | 信号 |
|---------------|----------|------|
| > 80 | 极高估值 | 泡沫风险 |
| 40-80 | 偏高估值 | 警惕 |
| 15-40 | 合理估值 | 正常 |
| < 15 | 低估值 | 安全边际高 |

#### 4. 持仓变动（风格漂移检测）

对比最近几个季度的持仓报告：
- 季度持仓变动 > 50% → 风格漂移严重
- 前十大重仓股每季大幅变动 → 策略不稳定
- 持仓行业从A行业突然切换到B行业 → 警惕

#### 5. 资产配置

查看股票/债券/现金的比例：
- 股票占比 > 80% → 激进型（股票型基金）
- 股票占比 40-80% → 均衡型（混合型基金）
- 股票占比 < 40% → 保守型
- 现金占比突然大幅增加 → 基金经理可能在减仓避险

## 四、Performance 规模与持有人结构

### 数据来源

```
https://fundf10.eastmoney.com/gmbd_{code}.html   # 规模变动
https://fundf10.eastmoney.com/cyrgm_{code}.html  # 持有人结构
```

### 规模分析

#### 1. 规模风险

| 基金规模 | 风险 | 操作 |
|----------|------|------|
| < 5000 万 | 清盘风险 | 立即赎回（铁律 4） |
| 5000 万 - 2 亿 | 迷你基金，关注 | 警惕 |
| 2-200 亿 | 适中 | 正常 |
| > 200 亿 | 巨无霸，调仓困难 | 警惕业绩下滑 |

#### 2. 规模变动趋势

- 规模连续 3 季度缩水 → 资金出逃，警惕
- 规模季度暴增 > 100% → 资金涌入过快，"规模魔咒"
- 规模稳定或缓慢增长 → 健康

**规模魔咒解释：基金规模突然暴增后，基金经理被迫买入更多股票，但好股票数量有限，只能买次优选择，导致业绩下滑。**

### 持有人结构

| 机构持有比例 | 含义 | 信号 |
|-------------|------|------|
| > 80% | 机构高度集中 | 踩踏风险——机构一旦赎回，净值暴跌 |
| 30-60% | 健康分散 | 机构认可且不集中，加分 |
| < 10% | 几乎全散户 | 散户主导，情绪化申赎导致波动 |

## 五、Price 费率分析

### 数据来源

```
https://fundf10.eastmoney.com/jjfl_{code}.html
```

### 费率类型

| 费率 | 说明 | 影响 |
|------|------|------|
| 申购费 | 买入时一次收取（通常 0.15%-1.5%） | 短期持有影响大 |
| 赎回费 | 卖出时收取，按持有时间递减 | <7天有1.5%惩罚性费率 |
| 管理费 | 每年收取（股票型通常 1.5%） | 长期持有主要成本 |
| 托管费 | 每年收取（通常 0.25%） | 固定成本 |
| 销售服务费 | C类份额每年收取（通常 0.4-0.8%） | 替代申购费 |

### A 类 vs C 类选择

**这是投资者最常困惑的问题，必须给出明确建议：**

| 持有时间 | 推荐 | 理由 |
|----------|------|------|
| > 1 年 | A 类 | A类一次性申购费，之后只有管理费；长期总成本更低 |
| 6个月-1年 | A 类 | 勉强A类更划算 |
| < 6 个月 | C 类 | C类无申购费，有销售服务费；短期总成本更低 |
| < 7 天 | 都别买 | 惩罚性赎回费 1.5%！ |

**比喻：A类像"年卡"——一次性付钱但后续便宜；C类像"月卡"——不收入场费但按月收费。**

### 费率竞争力评估

- 同类基金费率排名前 20%（最贵）→ 长期持有成本高
- 同类基金费率排名后 20%（最便宜）→ 长期持有省钱

**注意：指数基金费率应重点比较，因为被动管理基金的核心竞争力就是低成本。好的指数基金管理费应 < 0.5%。**

## 六、基金评级参考

### 数据来源

```
https://fundf10.eastmoney.com/jjpj_{code}.html
```

### 四大评级机构

| 机构 | 方法论 | 特点 |
|------|--------|------|
| 晨星（Morningstar） | 风险调整后收益 MRAR | 全球最权威 |
| 招商证券 | CAPM模型+经风险调整收益 | 偏学术 |
| 上海证券 | 选证能力+择时能力+夏普比率 | 三维评估 |
| 济安金信 | 多目标规划+类内多因素 | 最复杂 |

### 如何使用评级

1. **优先选 4-5 星基金**
2. **综合多家评级**——如果多家都给高分，可信度高
3. **注意评级滞后性**——评级 1-3 个月更新一次，不能反映最新变化
4. **评级要求基金成立满 3 年**——新基金没有评级不代表差

## 七、基金对比方法论

当用户持有多只同类基金时，按以下步骤对比：

### 步骤

1. **确认同类**：确保对比的基金是同一类型（如都是混合型），不同类型不可比
2. **同时间维度**：对比相同时间段的指标（如近 1 年）
3. **逐项打分**：按权重逐项比较
4. **给出建议**：保留评分更高的，合并或换出评分较低的

### 对比权重

| 维度 | 权重 | 数据来源 |
|------|------|----------|
| 综合评分（技术+专业指标） | 35% | 指标计算 |
| 风险调整后收益（夏普+卡玛） | 25% | 指标计算 |
| 费率 | 15% | 费率页面 |
| 基金经理稳定性 | 15% | 经理页面 |
| 规模与持有人结构 | 10% | 规模页面 |

### 输出格式

```
【同类基金对比】
                    基金A(代码)    基金B(代码)
  综合评分:          XX/100         XX/100
  夏普比率:          X.XX           X.XX
  最大回撤:          X%             X%
  年化收益:          X%             X%
  费率:              X%             X%
  经理任职:          X年            X年
  规模:              XX亿           XX亿
  
  建议: [保留A换出B / 均可保留 / 均建议减仓]
  理由: [具体原因]
```
