## 这是什么

「基金买卖圣经法典」是一个 **AI Skill**（技能文件），不是独立的应用程序。任何支持 Skill 加载的 AI Agent（如 opencode、Claude Code 等）加载此技能后，即可化身专业的基金分析专家，按照法典的 138 条信号规则 + 6 条铁律体系进行基金买卖分析。

**核心理念：规则就是规则，没有例外。不执行规则 = 亏损。数据必须实时，过期数据 = 错误判断。**

## 功能特性

### 七层分析体系（138 条信号规则）

| 层级 | 权重 | 规则数 | 覆盖内容 |
|------|------|--------|----------|
| 技术面 | 30% | 55 条 | MA/EMA、MACD/DMI/SAR、KDJ/RSI/WR/CCI、BOLL 布林带（1.5x 标准差）、BIAS 乖离率、OBV/VR 能量潮 |
| 专业指标 | 20% | 13 条 | Alpha、Beta、夏普比率、索提诺比率、卡玛比率、信息比率、跟踪误差、换手率 |
| 基本面 | 15% | 28 条 | 晨星五支柱（People/Parent/Process/Performance/Price）、持仓分析、规模与结构、费率分析 |
| 资金面 | 15% | 19 条 | 主力资金流向、主力伪装识别（拆单/对倒/虚假托单）、场外基金替代指标 |
| 宏观面 | 10% | 17 条 | 美联储利率、美元指数、A 股大盘、地缘政治与新闻面 |
| 共振信号 | 10% | 6 条 | BOLL+KDJ+RSI 三重共振（胜率 85%+）、MACD 背离共振、趋势三重确认 |

### 6 条不可违反的铁律

1. **双顶 + 颈线** → 减仓/清仓
2. **支撑跌破 3 日不回** → 认错离场
3. **基金经理更换** → 立即重新评估
4. **清盘风险**（规模 < 5000 万）→ 立即赎回
5. **市场状态检测** → 开盘 30 分钟内禁止分析（防止量价失真）
6. **实时数据强制** → 禁止使用缓存/历史/估算数据

### 全量技术指标

| 分类 | 指标 |
|------|------|
| 均线系 | MA5/10/20/60/120/250、EMA |
| 趋势系 | MACD（DIF/DEA/柱）、DMI（+DI/-DI/ADX）、SAR（抛物线） |
| 震荡系 | KDJ（K/D/J）、RSI（6/12/14 日）、WR（威廉 %R）、CCI（顺势） |
| 通道系 | BOLL（上轨/中轨/下轨，基金专用 1.5 倍标准差） |
| 偏离系 | BIAS（6/12/24 日乖离率） |
| 能量系 | OBV（能量潮）、VR（容量比率）— ETF 适用 |
| 风险系 | 最大回撤、年化波动率、夏普/索提诺/卡玛比率 |

### 资金面分析（主力密码功能）

- 主力资金流向（超大单/大单/中单/小单分类）
- 主力伪装识别（拆单、对倒、虚假托单/压单）
- 主力意图判断（吸筹、派发、洗盘、出逃）
- 场外基金替代指标（季度申赎规模、机构持有比例变化）

### 新手教学模式

- 85+ 术语大白话解释
- 生活化比喻库（Alpha=全班平均80你考85多出的5分，BOLL=乒乓球的房间天花板和地板）
- 每份报告末尾自动附「今天学到了」板块

### 基金分类策略

| 类型 | 核心指标 | 仓位上限 |
|------|----------|----------|
| 股票型/混合型 | Alpha + 持仓 + 技术面 | 30% |
| 指数型/ETF 联接 | 跟踪误差 + 费率 + 趋势 | 宽基 40% / 行业 20% |
| QDII | 美元 + 海外指数 + 汇率 | 30% |
| 债券型 | 利率 + 信用 + 夏普 | 50% |
| 货币型 | 七日年化 + 规模 | 按需 |

## 快速开始

### 安装

将本仓库克隆到你的 AI Agent 技能目录：

```bash
# opencode 用户
git clone https://github.com/windmills/fund-bible.git ~/.opencode/skills/fund-bible

# Claude Code 用户
git clone https://github.com/windmills/fund-bible.git ~/.claude/skills/fund-bible
```

### 使用

在你的 AI 对话中提到基金相关话题，Agent 会自动加载此技能：

```
用户：帮我分析一下 002112 德邦鑫星
Agent：[自动加载 fund-bible skill，执行七层分析，输出完整报告]
```

或手动指定：

```
/fund-bible 分析 002112
```

### 指标计算工具

内置 Python 脚本，可独立运行计算全部技术指标：

```bash
# 从东方财富 API 获取数据并计算
python3 scripts/indicators.py --code 002112

# 传入自定义净值数据
python3 scripts/indicators.py --navs 1.0,1.02,1.01,1.03,1.05
```

输出示例：
```
【均线系】MA5:6.02 MA10:6.06 MA20:5.93 MA60:5.07
【趋势系】MACD: DIF=0.21 DEA=0.25 柱=-0.07 [绿柱]
【震荡系】KDJ: K=55 D=52 J=62 [中性]  RSI(14): 55 [中性]
【通道系】BOLL: 上轨=6.31 中轨=5.93 下轨=5.54 带宽=13%
【风险系】最大回撤: 19.6%  夏普: 6.53  卡玛: 17.76
```

## 数据来源

所有数据通过东方财富网（EastMoney）公开 API 获取：

| 数据 | 来源 |
|------|------|
| 基金历史净值 | `fund.eastmoney.com/f10/F10DataApi.aspx` |
| 基金持仓/经理/费率 | `fundf10.eastmoney.com/` |
| 资金流向 | `push2.eastmoney.com/api/qt/stock/fflow/` |
| 大盘指数 | `push2his.eastmoney.com/api/qt/stock/kline/` |
| 宏观数据 | 美联储官网 + 东方财富全球指数 |

## 与 trading-agent 的区别

| 维度 | fund-bible（本技能） | trading-agent |
|------|---------------------|---------------|
| 分析对象 | 基金（净值曲线） | 股票/板块（实时行情） |
| 技术指标 | 全量 7 大类 20+ 指标 | MA/MACD/RSI 三种 |
| 基本面 | 晨星五支柱 + 持仓 + 费率 | 无 |
| 宏观面 | 美联储/美元/地缘政治 | 无 |
| 资金面 | 主力密码 + 伪装识别 | 基础资金流 |
| BOLL 参数 | 1.5 倍标准差（基金适配） | 2 倍标准差（股票） |
| 新手教学 | 85+ 术语 + 比喻库 | 基础术语 |
| 基金分类 | 5 类差异化策略 | 无（统一规则） |

两者完全独立，可同时加载，各管各的领域。
