# 技术分析详解

本文档详细说明每个技术指标在基金净值曲线上的判读规则。当 SKILL.md 中的规则表需要更深入解释时，参考本文档。

---

## 一、均线系（MA/EMA）

### 基本原理

均线（Moving Average）是过去 N 天基金净值的平均值连成的线。它平滑了日常波动，反映趋势方向。

**基金净值只有单点数据（每天一个值），均线计算方式与股票完全一致，直接用每日净值代替收盘价。**

### 关键均线含义

| 均线 | 周期 | 含义 | 比喻 |
|------|------|------|------|
| MA5 | 5个交易日 | 一周平均 | 短期体温计 |
| MA10 | 10个交易日 | 两周平均 | 短期趋势线 |
| MA20 | 20个交易日 | 一个月平均 | 月度趋势（最常用） |
| MA60 | 60个交易日 | 一个季度平均 | 中期趋势线 |
| MA120 | 120个交易日 | 半年平均 | 半年线（牛熊分界） |
| MA250 | 250个交易日 | 一年平均 | 年线（长期趋势） |

### 多头排列与空头排列

- **多头排列**：MA5 > MA10 > MA20 > MA60，所有均线向上
  - 含义：短期、中期、长期趋势全部向上，最强买入信号
  - 操作：持有或加仓

- **空头排列**：MA5 < MA10 < MA20 < MA60，所有均线向下
  - 含义：短期、中期、长期趋势全部向下，最强卖出信号
  - 操作：空仓观望

### 金叉与死叉

- **金叉**：短期均线（如 MA5）从下方上穿长期均线（如 MA20）
  - 含义：短期力量转强，趋势可能向上
  - 操作：关注/建仓

- **死叉**：短期均线从上方下穿长期均线
  - 含义：短期力量转弱，趋势可能向下
  - 操作：减仓

### 均线粘合

当 MA5、MA10、MA20 三条均线之间的间距小于 2%时，称为"粘合"。

- 粘合 = 多空力量均衡 = 变盘在即
- 粘合后向上发散 → 主升浪，跟进
- 粘合后向下发散 → 主跌浪，离场

## 二、趋势系（MACD / DMI / SAR）

### MACD（指数平滑异同移动平均线）

MACD 是最常用的趋势指标，由三部分组成：

| 组件 | 计算 | 含义 |
|------|------|------|
| DIF（快线） | EMA12 - EMA26 | 短期与长期趋势的差值 |
| DEA（慢线） | DIF 的 9 日 EMA | DIF 的平滑版 |
| MACD 柱 | 2 × (DIF - DEA) | 多空力量差 |

**判读规则：**

1. **DIF 上穿 DEA（金叉）**→ 趋势转多，买入信号
2. **DIF 下穿 DEA（死叉）**→ 趋势转空，卖出信号
3. **红柱（柱 > 0）**→ 多头力量强
4. **绿柱（柱 < 0）**→ 空头力量强
5. **红柱放大**→ 多头加速，持有
6. **红柱缩短**→ 多头减弱，警惕
7. **绿柱放大**→ 空头加速，空仓
8. **绿柱缩短**→ 空头减弱，关注反弹

**顶背离（最重要卖出信号）：**
净值创出新高，但 MACD 的 DIF 没有创出新高 → 上涨动能衰竭 → 先减为敬

**底背离（重要买入信号）：**
净值创出新低，但 MACD 的 DIF 没有创出新低 → 下跌动能衰竭 → 关注企稳

### DMI（趋向指标）

| 组件 | 含义 |
|------|------|
| +DI | 多头方向指标，上升代表多头力量增强 |
| -DI | 空头方向指标，上升代表空头力量增强 |
| ADX | 趋势强度指标，与方向无关 |

**判读规则：**
- +DI > -DI → 多头占优
- -DI > +DI → 空头占优
- ADX > 25 → 趋势明显（无论方向）
- ADX < 20 → 无明显趋势（震荡市）
- +DI > -DI + ADX 上升 → 强上升趋势，持有

### SAR（抛物线转向指标）

SAR 像抛物线一样的点，在净值上方或下方：

- **SAR 在净值下方（红色）**→ 多头，持有
- **SAR 在净值上方（绿色）**→ 空头，离场
- **SAR 翻转**（从红变绿或从绿变红）→ 趋势反转信号

## 三、震荡系（KDJ / RSI / WR / CCI）

### KDJ（随机指标）

KDJ 用近 N 日最高价、最低价和当前价格的关系来判断超买超卖。

**基金适配**：基金没有最高价/最低价，用近 N 日的最高净值和最低净值替代。

| 线 | 计算 | 含义 |
|----|------|------|
| K | RSV 的平滑值，0-100 | 短期超买超卖 |
| D | K 的平滑值，0-100 | 中期超买超卖 |
| J | 3K - 2D，可超出 0-100 | KDJ 的加速度 |

**判读规则：**
- K < 20 → 超卖区（跌太多了）
- K > 80 → 超买区（涨太多了）
- K 上穿 D（金叉），且 K < 20 → 低位金叉，买入信号
- K 下穿 D（死叉），且 K > 80 → 高位死叉，卖出信号
- J < 0 → 极度超卖
- J > 100 → 极度超买

**注意：KDJ 在单边趋势中容易"钝化"（长时间停留在超买或超卖区），此时 KDJ 失效，需结合其他指标。**

### RSI（相对强弱指标）

RSI 比较一段时期内平均涨幅和平均跌幅，判断多空力量。

| RSI 值 | 含义 | 操作 |
|--------|------|------|
| > 80 | 极度超买 | 警惕卖出 |
| 70-80 | 超买区 | 关注 |
| 30-70 | 中性区 | 正常 |
| 20-30 | 超卖区 | 关注 |
| < 20 | 极度超卖 | 关注买入 |

**RSI 背离（高胜率信号）：**
- 顶背离：净值新高 + RSI 未新高 → 上涨乏力，减仓
- 底背离：净值新低 + RSI 未新低 → 下跌乏力，关注反弹

### WR（威廉指标 %R）

WR 衡量当前价格在近 N 日价格区间中的位置。

| WR 值 | 含义 |
|-------|------|
| > 80 | 超卖区（接近区间底部） |
| < 20 | 超买区（接近区间顶部） |

注意：WR 的超买超卖方向与 KDJ/RSI 相反——WR 高=超卖，WR 低=超买。

### CCI（顺势指标）

CCI 衡量价格与均值的偏离程度。

| CCI 值 | 含义 |
|--------|------|
| > +100 | 超买区 |
| < -100 | 超卖区 |
| 上穿 -100 | 超卖回升，买入信号 |
| 下穿 +100 | 超买回落，卖出信号 |

## 四、通道系（BOLL 布林带）

### 基本原理

布林带用均线 ± 标准差画出三条轨道，反映净值运行通道。

**基金专用参数：1.5 倍标准差（股票用 2 倍，但基金波动小，2 倍太宽松信号太少）**

| 轨道 | 计算 | 比喻 |
|------|------|------|
| 上轨（UP） | 中轨 + 1.5 × 标准差 | 天花板 |
| 中轨（MB） | 20 日均线 | 地面 |
| 下轨（DN） | 中轨 - 1.5 × 标准差 | 地板 |

**带宽（Bandwidth）**= (上轨 - 下轨) / 中轨 × 100%
- 带宽 < 5% → 收口（变盘在即）
- 带宽 > 15% → 开口（趋势启动）

### 判读规则

| 净值位置 | 含义 | 操作 |
|----------|------|------|
| 上轨上方 | 极度超买 | 警惕回调 |
| 上轨附近 | 超买区 | 警惕 |
| 中轨上方 | 偏强 | 持有 |
| 中轨附近 | 中性 | 观察 |
| 中轨下方 | 偏弱 | 谨慎 |
| 下轨附近 | 超卖区 | 关注反弹 |
| 下轨下方 | 极度超卖 | 关注反弹（非暴跌时） |

### 收口与开口

1. **收口**：带宽缩小，净值在窄幅波动 → 多空僵持，变盘在即 → 观望
2. **向上开口**：收口后上轨上翘，下轨下翘 → 趋势选择向上 → 跟进
3. **向下开口**：收口后上轨下翘，下轨上翘 → 趋势选择向下 → 离场

## 五、偏离系（BIAS 乖离率）

乖离率衡量净值偏离均线的程度。

**公式**：BIAS = (净值 - MA) / MA × 100%

| 指标 | 超买 | 超卖 | 含义 |
|------|------|------|------|
| BIAS6（6日） | > 5% | < -5% | 短期偏离 |
| BIAS12（12日） | > 10% | < -10% | 中期偏离 |
| BIAS24（24日） | > 15% | < -15% | 长期严重偏离 |

**原理**：均值回归——偏离太多就会拉回来，就像橡皮筋拉太长会弹回。

## 六、能量系（OBV / VR）

**仅适用于 ETF 联接基金（可获取底层 ETF 成交量）。场外基金无成交量数据。**

### OBV（能量潮）

OBV 累计净值的涨跌方向，反映"量能"趋势：

- OBV 上升 + 净值上升 → 量价配合，趋势健康
- OBV 下降 + 净值上升 → 量价背离，上涨乏力
- OBV 上升 + 净值下降 → 底部蓄力，关注反转

### VR（容量比率）

VR 衡量上涨日和下跌日的成交量比值：

- VR > 150 → 多头主导
- VR < 70 → 空头主导

## 七、风险系（基金专属）

### 最大回撤（Maximum Drawdown）

**定义**：从历史最高净值跌到最低净值的最大跌幅。

**公式**：最大回撤 = max[(峰值 - 谷值) / 峰值]

**含义**：回答"最惨的时候亏多少"。

| 回撤 | 风险等级 | 操作 |
|------|----------|------|
| < 10% | 低风险 | 正常配置 |
| 10-20% | 中等风险 | 适度配置 |
| 20-30% | 高风险 | 轻仓 |
| > 30% | 极端风险 | 不碰或极轻仓 |

### 年化波动率

净值日收益率的年化标准差。反映净值的震荡幅度。

| 波动率 | 风险等级 |
|--------|----------|
| < 15% | 低波动（债券型） |
| 15-25% | 中波动（混合型） |
| > 25% | 高波动（股票型/QDII） |

### 夏普比率（Sharpe Ratio）

每承担 1 单位风险获得的超额收益。

**公式**：夏普 = (年化收益 - 无风险利率) / 年化波动率

| 夏普 | 评价 |
|------|------|
| > 1.5 | 优秀 |
| 1.0-1.5 | 良好 |
| 0.5-1.0 | 一般 |
| 0-0.5 | 较差 |
| < 0 | 冒险还亏钱 |

### 索提诺比率（Sortino Ratio）

夏普比率的改良版——只把下跌视为风险，上涨不算风险。

**公式**：索提诺 = (年化收益 - 无风险利率) / 下行波动率

索提诺通常高于夏普（分母更小），> 2 算优秀。

### 卡玛比率（Calmar Ratio）

年化收益 / 最大回撤。衡量"用最小回撤换最大收益"。

**公式**：卡玛 = 年化收益 / 最大回撤

| 卡玛 | 评价 |
|------|------|
| > 3 | 优秀 |
| 1-3 | 良好 |
| < 1 | 较差 |

## 八、多指标共振策略

**核心思想：单个指标容易误判（KDJ 单用胜率仅 47%），多指标同时确认时胜率大幅提升（85%+）。**

### 买入共振

**三重超卖共振买入（RES-1）：**
1. BOLL：净值触及或跌破下轨
2. KDJ：K < 20 区域出现金叉
3. RSI：< 30
→ 三重确认，胜率 ~85%

**趋势反转买入（RES-3）：**
1. MACD 底背离（净值新低但 MACD 未新低）
2. BOLL 下轨企稳（触及下轨后不再下跌）
→ 趋势可能反转，胜率 ~75%

**三重趋势做多（RES-5）：**
1. MA 金叉（MA5 上穿 MA20）
2. MACD 金叉（DIF 上穿 DEA）
3. BOLL 向上开口
→ 趋势确认，胜率 ~80%

### 卖出共振

**三重超买卖出（RES-2）：**
1. BOLL：净值触及或突破上轨
2. KDJ：K > 80 区域出现死叉
3. RSI：> 70
→ 三重确认，胜率 ~85%

**趋势反转卖出（RES-4）：**
1. MACD 顶背离（净值新高但 MACD 未新高）
2. BOLL 上轨滞涨（触及上轨后连续 3 日不突破）
→ 趋势可能反转，胜率 ~75%

## 九、形态识别（基金净值曲线版）

基金净值是单点数据（无 OHLC），K 线形态（长上影、十字星等）不适用，但以下形态有效：

### 双顶（M头）

净值涨到高点 A，回落到颈线，再涨到高点 B（接近 A），再回落。

- **双顶形成** → 减仓 50%
- **颈线跌破** → 清仓

### 双底（W底）

净值跌到低点 A，反弹到颈线，再跌到低点 B（接近 A），再反弹。

- **双底形成** → 关注建仓
- **颈线突破** → 确认反转，建仓

### 平台突破

净值在某个区间横盘较长时间（>20 日），然后放量突破。

- **向上突破** → 最佳跟进点
- **向下突破** → 离场
